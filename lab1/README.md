# Модель зоопарка

Система автоматизации работы салона красоты, реализованная на Python с соблюдением принципов ООП, чистой архитектуры, стандартов PEP8, аннотации типов.

### Основные сущности

* Животное - класс `Animal`
* Вольер - `Enclosure`
* Мероприятие - `Event`
* Экспозиция - `Exposition`
* Кормление - `Feed`
* Генератор ID - `IDGenerator`
* Экскурсия - `Tour`
* Экскурсовод - `TourGuide`
* Ветеринар - `Vet`
* Запись ветеринара - `VetLog`
* Прохождение медосмотра - `VetService`
* Посетитель - `Visitor`
* Зоопарк - `Zoo`

### Реализованные поведения
* операция ухода за животными
* операция кормления
* операция ветеринарного обслуживания
* операция организации экскурсий
* операция проведения образовательных мероприятий
