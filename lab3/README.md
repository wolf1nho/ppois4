# Crimsonland Game

A simple top-down shooter game inspired by Crimsonland, built with Pygame.

## Installation

1. Install Python 3.x
2. Install dependencies: `pip install -r requirements.txt`
3. Run the game: `python3 main.py`

## Controls

- WASD (or Russian layout analogues): Move
- Mouse: Aim and shoot (left click)
- Q/E or 1-4: Switch weapon
- P or Space: Pause
- ESC: Back/Quit

## Features

- Improved crosshair and HP bar
- Multiple enemy types: normal, tank, swift and shooter
- Enemy projectiles and different combat stats
- Wave mode: 25 waves total, boss on every 5th wave
- Death visual effects
- Highscore table with player name, score, date and time
